
## Car Lane Game

Implemented using HTML, CSS, and Typescript




## Run Locally

Clone the project

```bash
  git clone https://github.com/Nis13/CarLane_game.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Run the project
```bash
  npm run dev
```
